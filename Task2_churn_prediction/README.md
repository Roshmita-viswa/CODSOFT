# 📊 ChurnSense - Customer Churn Prediction

ChurnSense is an AI-powered customer churn prediction system developed using Machine Learning and Streamlit.

This project predicts whether a customer is likely to leave a subscription-based service using customer demographics and banking behavior.

---

## 🚀 Features

- Professional Streamlit Dashboard
- Customer Churn Prediction
- Random Forest Classifier
- Risk Level Analysis
- Probability Analysis
- Customer Retention Suggestions
- Modern Dark UI

---

## 🛠 Technologies Used

- Python
- Pandas
- Scikit-learn
- Streamlit
- Joblib

---

## 📂 Dataset

Bank Customer Churn Dataset
